# Procédure de traduction d’une page

## Cas 1 — Traduction automatique via le bouton **Translate**

1. Cliquer sur le bouton **Translate**  
   (situé sous les boutons **Save** et **Publish**)

2. La traduction est créée automatiquement.

3. Accéder à la version anglaise de la page :
   - Cliquer sur la select box **French (fr)** en haut à droite.
   - Sélectionner **English (en)**.

4. Vérifier le contenu :
   - Si les champs sont bien traduits :
     - Vérifier que **tous les modules/composants** de la page sont correctement remplis.
     - Vérifier que **les liaisons** sont bien en place  
       (par exemple : `tag_type` pour un tag).
     - Cliquer sur :
       - **Publish** pour publier la page, ou
       - **Save** si vous ne souhaitez pas encore la publier.
   - Si **English (en)** n’est pas disponible **ou** si les champs ne sont pas traduits, passer au **Cas 2**.

---

## Cas 2 — Traduction automatique non créée ou champs non remplis

1. Passer sur la page en anglais :
   - Cliquer sur la select box **French (fr)** en haut à droite.
   - Sélectionner **English (en)**.
   - Si **English (en)** n’est pas disponible, cliquer sur **+ Create English (en) locale**.

2. Une fois sur la page en anglais :
   - Cliquer sur l’icône **globe** (map monde) en haut à droite, juste à côté de la select box de langue.

3. Dans le pop-up qui s’ouvre :
   - Sélectionner **French (fr)** dans la select box.
   - Cliquer sur **Yes, fill in**.

4. Résultat :
   - Tous les **champs**, **composants/modules** et **liaisons** sont automatiquement remplis avec les données françaises.
   - Le contenu peut maintenant être **traduit manuellement**.

5. Finalisation :
   - Vérifier les modules et les liaisons.
   - Cliquer sur **Publish** ou **Save** selon le besoin.
